# Analogy and Metaphor Control

Analogies and metaphors are among the most common ways AI writing inserts decoration
disguised as clarity. They sound vivid but often carry no information. The default is
no analogies. Write literally unless the permission test passes.

---

## Permission Test

Use an analogy only if all five conditions pass:

1. The subject is genuinely unfamiliar, abstract, or technical to this reader.
2. The analogy makes the idea easier to understand than a literal explanation would.
3. The analogy is shorter than the literal explanation.
4. The analogy is precise enough that it will not mislead the reader.
5. The sentence sounds normal when read aloud in conversation.

If any condition fails, write literally.

---

## Frequency Limits

| Text length | Analogy limit |
|---|---|
| Under 800 words | 0 by default |
| 800 to 1,500 words | 1 maximum, only if it passes the permission test |
| Over 1,500 words | 1 per 1,500 words maximum |

Never use more than one analogy in the same section. Never extend an analogy across
multiple paragraphs.

---

## Banned Analogy Setups

These trigger phrases almost always introduce a metaphor. Apply the permission test
immediately. If the test fails, delete the analogy and write literally.

- "Think of it as"
- "Imagine" / "Picture"
- "It is like" / "It's like" / "It is kind of like"
- "As if" / "As though"
- "The X of Y" (e.g., "the Airbnb of education")
- "Works like" / "Acts like" / "Functions as"
- "A bridge between"
- "A lens for" / "Through the lens of"
- "A mirror of"
- "A roadmap for"
- "The engine of" / "The fuel for"
- "The backbone of" / "The foundation of" (when metaphorical)
- "The fabric of"
- "The heartbeat of"
- "The DNA of"
- "The glue that holds"

---

## Banned Metaphor Families

These entire families are banned because AI uses them reflexively. Even when a specific
instance passes the permission test, check whether a literal description is cleaner.

| Family | Example to avoid | Write instead |
|---|---|---|
| Journey (for growth or progress) | "on this learning journey" | "as you learn this" |
| Battlefield (for work or competition) | "fighting for results" | "working toward results" |
| Machine (for people or teams) | "a well-oiled machine" | "a team that coordinates well" |
| Architecture (for ideas) | "the framework of thought" | "the approach" |
| Ecosystem (for organizations) | "our innovation ecosystem" | "our research network" |
| Engine / fuel (for motivation) | "what fuels our research" | "what drives our research" |
| Map / compass (for strategy) | "our strategic roadmap" | "our plan" |
| Signal / noise (unless literal) | "cutting through the noise" | "focusing on what matters" |
| Toolbelt / toolbox | "adding to your toolkit" | "learning another method" |
| Iceberg | "just the tip of the iceberg" | "there is more: [name it]" |
| Bridge (for connection) | "bridging theory and practice" | "connecting theory to practice" |
| North star (for goals) | "our north star metric" | "our primary metric" |
| Flywheel (for momentum) | "the flywheel effect" | "growth that compounds" |
| Scaffolding (for support) | "scaffolding student learning" | "supporting students step by step" |
| Gardening (for development) | "cultivating talent" | "developing staff" |
| Chess (for strategy) | "the opening gambit" | "the first move" |
| Sports (for performance) | "moving the goalposts" | "changing the criteria" |
| Puzzle (for complexity) | "piecing together the evidence" | "combining the evidence" |

## Banned Metaphor Verbs for Abstract Work

These verbs make writing sound vivid when applied to ideas, strategies, writing,
organizations, or emotions. They carry no specific information. Use literal verbs.

| Avoid | Use instead |
|---|---|
| woven | combined, connected, joined |
| layered | added, built on, stacked |
| carved out | created, made, established |
| baked in | built in, included from the start |
| fueled | driven by, caused by |
| sparked | started, triggered, caused |
| anchored | based on, grounded in |
| distilled | reduced to, simplified to, extracted from |
| unpacked | explained, analyzed, broken down |
| crystallized | clarified, became clear, took shape |
| sharpened | improved, refined, clarified |
| amplified | increased, strengthened |
| channeled | directed, focused, used |
| sculpted | shaped, formed, built |
| cemented | fixed, established, confirmed |
| bridged | connected, joined, linked |
| stitched together | combined, assembled |
| sanded down | smoothed, simplified, reduced |
| stripped back | simplified, reduced, removed |
| bolted on | added, attached, appended |

## Analogy Audit (Quick Check)

Before delivering prose, scan for these words. If found, apply the permission test.
Delete the analogy if the test fails.

- like / as if / as though
- imagine / picture
- kind of like / works like / acts like / functions as
- lens / bridge / roadmap / engine / fuel
- foundation / fabric / glue / heartbeat / DNA
- through the lens of / serves as (when metaphorical)

---

## Rewrite Examples

**Bad:** "Assessment is a lens for understanding student thinking."
**Better:** "Assessment shows how students think."

**Bad:** "The course scaffolds student learning from basic to advanced concepts."
**Better:** "The course moves from basic to advanced concepts."

**Allowed (passes test):** "Think of voltage as pressure in a pipe: the higher the pressure, the more current flows through the resistance." The subject is technical, the analogy is precise, it is shorter than a formal explanation, and it will not mislead the reader.

**Bad:** "Research weaves together quantitative and qualitative strands."
**Better:** "The research combines quantitative and qualitative data."

**Bad:** "This paper distills three years of fieldwork into actionable insights."
**Better:** "This paper summarizes three years of fieldwork."
